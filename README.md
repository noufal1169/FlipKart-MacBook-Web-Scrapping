# FlipKart-MacBook-Web-Scrapping
Collected data on macbooks available in flipkart

## Programming Language 
PYTHON3

## Modules used
1. Requests
2. Beautiful soup
3. time
4. pandas

## Collected data includes:
1. Product Name
2. Price
3. Ratings
4. Warrenty
5. Number of Ratings
6. Number of Reviews
7. Features
